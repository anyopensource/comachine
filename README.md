# Comachine
Finite-State Machine for Kotlin coroutines

# Binaries
```groovy
// in project build file
allprojects {
    repositories {
        mavenCentral()
        maven { url = "https://oss.sonatype.org/content/repositories/snapshots/" }
    }
}

// in module build file
dependencies {
    implementation 'de.halfbit:comachine-jvm:1.0-SNAPSHOT'
}
```
